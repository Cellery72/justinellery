Favourite Technologies

HTML5

With HTML being the core of every website, HTML5 has brought developers a plethora of new tools to utilize and master. The long overdue update has given developers a chance to build modern websites with features you'd expect to see in this day in age.

Bootstrap

Bootstrap's mobile first approach has gained a lot of headway in the world of development. Never has it been easier to make websites look great on a variety of different devices.

Visual Studio

Being a student has big advantages when it comes to software. Visual Studio has become my IDE of choice for developing almost everything from C++/C# to HTML/CSS. Microsoft has done a wonderful job of integrating the best features year after year with each new release.

Version Control

Learning the hard way is never easy, version control plays a vital role in software projects. Git and GitHub have quickly become my favourite type of Version Control Software for all of my projects.

CSS3

Casscading Style Sheets have always played a large part in web development, mainly because they are the core of every beautiful website. With the introduction of CSS3 and the full functioning backwards compatability developers can do more than ever to make great sites come to life.

Adobe Creative Suite

For years Adobe has been producing the best set of design tools available to web and graphics designers alike. With the help of Photoshop, Illustrator, Bridge and more it's become increasingly easier to produce high quality logos, vectors and designs in the blink of an eye.

SEO

With over a billion websites online right now and growing by the thousands per day, you need to stand out. You need to be visible to potential customers and clients. Search Engine Optimization plays a large part in being a succesful buisness.

Wordpress

I've grown a strong love for Wordpress when it comes to nearly anything these days. Whether it be with the power of building scalable e-commerce websites, to basic 4 page sites or even simply blogging it's become my favourite CMS.
